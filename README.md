<h1 align="center">
  Movie App
</h1>

## Features

- Splash Screen
- Movies Now Playing
- Popular Movies
- Top Rated Movies
- Upcoming Movies
- Search Movies

## Installation

```sh
$ git clone https://github.com/kingkinfajarr/movie-app.git
$ cd movie-app
$ flutter pub get
$ run project with CRTL + F5 or flutter run
```
